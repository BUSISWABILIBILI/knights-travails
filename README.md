# Knights Travails

A JavaScript solution for finding the shortest path a knight can take between two squares on a chessboard.

## Features

- Represents the chessboard as an implicit graph
- Uses Breadth-First Search
- Finds the shortest possible path
- Validates board positions
- Prints the number of moves
- Prints each square in the path

## Example

```javascript
printKnightPath([3, 3], [4, 3]);
```
